# Chunsik Dodge

3D 미사일 회피 게임. 플레이어가 아레나 위에서 떨어지는 미사일을 피하며 생존한다.

## Language

**개인전 (Solo)**:
1인 플레이 모드. `GameMode = 'solo'`.
_Avoid_: 1P 모드, 싱글

**대결전 (Versus)**:
2인 플레이 모드. `GameMode = 'versus'`. 캐주얼 표현으로 "전투모드"라고도 부르지만, 코드·UI의 정식 용어는 항상 "대결전 / versus"다.
_Avoid_: 전투모드, 배틀모드, VS 모드, PvP

**아레나 (Arena)**:
미사일 회피가 일어나는 직사각형 플레이 영역. 폭·깊이·플레이어 반지름 등의 공간 파라미터로 정의되며, 한 게임 세션 동안 정확히 하나의 인스턴스가 존재한다.
_Avoid_: 필드, 스테이지

**맵 (Map)**:
**아레나**의 크기 프리셋 + 그 크기에 비례해 그려지는 시각적 부속(바닥, 레일, 그리드)을 한 묶음으로 부르는 이름. 게임플레이 규칙(미사일 스폰 간격·속도·종류, 페이즈 진행 등)은 **맵**에 종속되지 않으며, 난이도 축은 별도(`difficulty.ts`).
_Avoid_: 스테이지, 필드, 레벨

**일반맵 (Normal Map)**:
현재 `ARENA = 17×11` 크기를 사용하는 기본 **맵**. **개인전**의 유일한 선택지이자, **대결전**의 기본 선택지.

**확장맵 (Extended Map)**:
**일반맵**보다 큰 크기를 갖는 **맵**. 치수 `24 × 15.5` (현재 비율 ≈ 1.55:1 유지, 선형 1.4×, 면적 ≈ 2×). **대결전**에서만 선택 가능.

**맵 프리셋 (Map Preset)**:
`config.ts`에 선언된 **맵**의 명세. `MAP_PRESETS: Record<'normal' | 'extended', ArenaSpec>` 형태. `Game` 인스턴스가 게임 시작 시 활성 프리셋을 `this.arena`에 캐시한다.

## Relationships

- **개인전**과 **대결전**은 모두 **아레나** 위에서 진행된다
- **아레나**의 크기는 선택된 **맵**으로 결정된다
- **개인전**은 항상 **일반맵**을 사용한다
- **대결전**은 **일반맵** 또는 **확장맵**을 선택할 수 있다 (미구현)
- **맵** 선택은 메뉴(`state === 'ready'`)에서만 가능 — 진행 중 전환 불가
- **맵** 선택값은 `STORAGE_KEYS.versusMap`에 영속화 (`'normal' | 'extended'`, 기본 `'normal'`)
- **맵**은 난이도/스폰 규칙에 영향을 주지 않는다 — 공간 크기와 시각 표현만 결정한다
- 미사일 스폰 마진·정리 거리는 절대 상수로, **맵** 크기와 독립적이다

## Flagged ambiguities

- 사용자가 "전투모드"라는 단어를 사용한 적 있음 — 해소: **대결전(versus)**의 캐주얼 표현일 뿐, 코드·UI·문서는 모두 "대결전"으로 유지

## Known unknowns

- **확장맵 + 모바일** 조합의 플레이 경험은 미검증. 카메라가 더 멀어지면 모바일 화면에서 캐릭터가 작아질 수 있음. 1차 구현 후 튜닝 필요 여부 판단.
- **확장맵 + 미사일 스폰 빈도** 튜닝. 면적 2배에 동일 스폰 규칙이면 체감 난이도가 낮아질 수 있음. 의도된 트레이드오프지만 사용자 피드백 후 재검토 가능.
